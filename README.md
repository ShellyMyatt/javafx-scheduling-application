# JavaFX Scheduling Application

## Overview

This project is a desktop scheduling application built using JavaFX and MySQL. It allows users to manage customers and appointments, including creating, updating, and deleting records.

## Features

* User login authentication
* Add, update, and delete customers
* Add, update, and delete appointments
* Appointment scheduling with time zone handling
* Reports showing appointment data by type and month
* Contact schedules with detailed appointment information

## Technologies Used

* Java (JDK 17)
* JavaFX
* MySQL
* JDBC
* Maven

## Key Concepts Demonstrated

* MVC architecture
* Database integration using SQL
* Data validation and error handling
* Time zone conversion (UTC, local, EST)
* Object-oriented programming principles

## How to Run

1. Clone the repository
2. Open the project in IntelliJ
3. Set up the MySQL database
4. Run the application from the Main class
