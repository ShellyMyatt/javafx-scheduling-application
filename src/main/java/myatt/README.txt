How to run the program:
Launch the application. The application starts by loading the LoginScreen.fxml file. The user must enter a valid
username and password from the database.
For this project a test user is created:
username: test
password: test
Once logged in the user will be alerted if they have an appointment in the next 15 minutes. The user will be directed
to the Appointment screen where they have the option to Add, Modify and Delete an Appointment.
The user can view the monthly or weekly appointments by selecting the corresponding radio buttons.
The user can navigate to the Customer Information Screen, Reports screen or Sign out of the application.
If the user navigates to the Customer Information Screen they have the option to Add, Modify and Delete Customers.
From the Customer Information Screen the user can navigate back to the Appointments screen by selecting the Main Menu button.
If the user navigates to the Reports screen they will be able to view the reports.
From the Reports screen the user can navigate back to the Appointments Screen or to the Customer Information Screen.
To exit the application the Sign-Out button on the Application Screen can be clicked.

Description of the additional report in part A3f: Total Customers By Country report shows the total number of customers in each country.
